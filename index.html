BIS-supported products
Documentation, integration guides, and API references for each product.
EFM

Electronic Filing Manager

Tyler's court e-filing backbone. Handles submissions, routing, and processing for CMS and EFSP integrations. Covers certificate process, onboarding, and error resolution.

Certificates EFSP CMS

re:Search

re:Search

Court record search and document retrieval API. Supports GetCase, GetDocument, and NotifyCaseEvent across ECF and Batch modes with configurable security overrides.

Press Review Tool

Managed media access for court records. Enables secure press review workflows with EFSP token authentication, SoapUI testing support, and SQL-based diagnostics.
